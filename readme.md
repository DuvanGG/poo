# Proyecto POO en Java

Este repositorio contiene modelos de dominio en Java para practicar Programación Orientada a Objetos (POO). Incluye clases simples como Persona, Fruta, CuentaBancaria, Customer, Product y Order. combinando el uso de atributos y métodos public, protected y private.

## Estructura del proyecto
- app/
  - bin/main/org/poo/
  - src/main/java/org/poo/
    - Persona.java
    - Fruta.java
    - CuentaBancaria.java
    - Customer.java
    - Product.java
    - Order.java

## Tecnologías y dependencias
- Java JDK (versión 21).
- Gestor de dependencias Gradle.

## Clases principales y su propósito
- Persona: modelo de persona con nombre, apellidos, fecha de nacimiento y altura.
- Fruta: modelo de fruta con nombre, peso promedio y colores.
- CuentaBancaria: modelo de cuenta bancaria simple con número de cuenta y estado.
- Customer: cliente del sistema con identificador, nombre, email, passwordHash y direcciones asociadas.
- Product: producto del catálogo con id, nombre, precio, stock e internalSku.
- Order: orden de compra con id, total, estado y token de pago.
